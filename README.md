# React CW

Coursework React application with:

- `HelloWorld.jsx` for basic JSX rendering
- `Counter.jsx` for state and interaction
- `List.jsx` for rendering arrays with `map()`
- `FilteredList.jsx` for search and category filtering
- `ReflectiveResponses.md` for the written reflection

## Run locally

```bash
npm install
npm start
```

## Test

```bash
npm test -- --watchAll=false
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This repo is configured for GitHub Pages at:

`https://mosakaa.github.io/React_CW`

Deploy with:

```bash
npm run deploy
```
